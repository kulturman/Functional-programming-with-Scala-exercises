# Functional Programming with Scala — Exercises Solutions

This repository contains my solutions to the exercises from the book **Functional Programming in Scala**.

## Purpose

The goal of this repo is to:

- practice functional programming concepts in Scala,
- keep a clear record of exercise solutions,
- track progress chapter by chapter.

## Book

- **Title:** Functional Programming in Scala
- **Authors:** Paul Chiusano and Rúnar Bjarnason

## Repository structure

Solutions are organized by chapter, with each chapter containing the corresponding exercise implementations.

```text
.
├── chapter-01/
├── chapter-02/
├── chapter-03/
└── ...
```

> The exact folder names may evolve as solutions are added.

## Status

This repository is a work in progress and will be updated as I continue solving exercises.

## Notes

- Solutions are primarily written for learning and practice.
- Some exercises may have more than one valid approach; the implementations here reflect my current understanding.
